# Slack Integration

This document explains how you can set up a slack application, and the required steps in the frontend in order to create the slack application for the setup of this repository.

## Initial Setup

Navigate to `https://api.slack.com/apps`. You may need to log in, and you may need to apply for the developer capabilities.

You will also need access to a slack organization in which you are an admin. Relevant guides for this can be found online.
